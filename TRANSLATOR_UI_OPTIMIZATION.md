# 在线翻译界面布局优化

## 优化目标

当前样式占据了大部分垂直空间，导致翻译文本输入/输出区域太小。

## 优化内容

### 1. 缩减顶部工具栏 (translator-toolbar)

**修改前**：
- padding: 1.5rem
- 标题字体: 1.25rem
- 显示副标题
- 垂直布局（标签在上，下拉框在下）

**修改后**：
- padding: 1rem → 0.75rem（减少 50%）
- 标题字体: 1.1rem（减少 12%）
- **隐藏副标题**（节省一行空间）
- 水平布局（标签和下拉框在同一行）

**节省空间**: ~40px

---

### 2. 压缩语言选择栏 (translator-langbar)

**修改前**：
- margin-top: 1.25rem
- 下拉框高度: 2.5rem
- padding: 0.5rem
- 切换按钮: 2.75rem × 2.5rem

**修改后**：
- margin-top: 0.75rem（减少 40%）
- 下拉框高度: 2rem（减少 20%）
- padding: 0.35rem（减少 30%）
- 切换按钮: 2.25rem × 2rem（缩小）

**节省空间**: ~30px

---

### 3. 优化文本区域 (translator-grid)

**修改前**：
- gap: 1rem
- margin-top: 1.25rem
- min-height: 240px（固定）
- padding: 1rem

**修改后**：
- gap: 0.75rem（减少 25%）
- margin-top: 0.75rem（减少 40%）
- **自适应高度**（flex: 1，填充剩余空间）
- padding: 0.75rem（减少 25%）
- resize: none（禁止手动调整，避免破坏布局）

**节省空间**: ~20px + 自适应

---

### 4. 压缩按钮区域 (translator-actions)

**修改前**：
- margin-top: 1rem
- gap: 0.75rem
- 按钮 min-width: 120px
- 按钮 padding: 默认

**修改后**：
- margin-top: 0.75rem（减少 25%）
- gap: 0.5rem（减少 33%）
- 按钮 min-width: 100px（减少 17%）
- 按钮 padding: 0.5rem 1rem
- 按钮字体: 0.9rem

**节省空间**: ~15px

---

### 5. 优化主布局

#### content-header（页面顶部）

**修改前**：
- padding: 1.5rem 2rem
- h1 字体: 默认（约 2rem）

**修改后**：
- padding: 0.875rem 1.5rem（减少 42%）
- h1 字体: 1.25rem
- user-info 字体: 0.9rem

**节省空间**: ~20px

#### content-area（内容区域）

**修改前**：
- padding: 2rem

**修改后**：
- padding: 1rem（减少 50%）
- 添加 flex: 1 和 overflow-y: auto
- 使其能充分利用剩余空间

#### translator-shell（翻译器容器）

**修改前**：
- padding: 1.5rem
- max-height: calc(100vh - 120px)

**修改后**：
- padding: 1rem（减少 33%）
- **height: calc(100vh - 140px)**（固定高度，填充整个可用区域）
- display: flex, flex-direction: column

## 总体效果

### 空间节省

| 区域 | 节省空间 |
|------|---------|
| 顶部工具栏 | ~40px |
| 语言选择栏 | ~30px |
| 文本区域边距 | ~20px |
| 按钮区域 | ~15px |
| 页面 header | ~20px |
| 内容区域 padding | ~30px |
| **总计** | **~155px** |

### 文本区域增加

由于采用 flexbox 自适应布局：
- 原来固定 240px
- 现在: `calc(100vh - 约380px)`
- 在 1080p 屏幕（1920×1080）：约 **700px**
- **增加约 460px（+192%）**

## 代码变更

修改文件：
- `app/static/css/admin.css`

主要变更：
1. `.translator-shell`: 固定高度 + flexbox
2. `.translator-toolbar`: 水平布局，隐藏副标题
3. `.translator-langbar`: 缩小间距和控件
4. `.translator-grid`: flex: 1 自适应
5. `.translator-textarea`: resize: none + flex: 1
6. `.content-header`: 缩小 padding 和字体
7. `.main-content`: flexbox 布局
8. `#content-area`: 缩小 padding + flex: 1

---

1. **不同分辨率测试**
   - 1920×1080 (主流)
   - 1366×768 (小屏幕)

3. **浏览器兼容性**
   - Chrome/Edge (Chromium)
   - Firefox
   - Safari

## 效果总结

✅ **操作区域增加 ~192%**（从 240px → 700px）  
✅ **界面更紧凑**（节省 ~155px 垂直空间）  
✅ **视觉更清爽**（减少无用空白）  
✅ **完全响应式**（自适应屏幕高度）  
✅ **保持可读性**（字体大小合理）
